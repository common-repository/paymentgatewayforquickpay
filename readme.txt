== Description ==

You are able to integrate your QuickPay payment gateway plugin  with Jomres install. With QuickPay, you can pay by credit/debit cards in safe way.
QuickPay is most widely used payment gateway to process payments online and accepts Visa, MasterCard, JCB and other variants of cards.

== Installation ==


1> In order to install the plugin in jomres, you have to click the plugin manager of jomres and then click on the install third party plugin 

2> Choose the plugin then click on install button

3> Now you have to click on Gateways of payment methods of jomres

4> Now Click on Edit button of QuickPay plugin

5> Now Activate this plugin and enter the required details.

   Merchant Id :- It is generated by QuickPay
   
   Agreement Id :- It is generated by QuickPay
   
   Payment Window Api key :- It is generated by QuickPay
   
6> After that click on jomres link of admin side
   
7> Now you have to click on Dashboard button of property and then click the property configuration of settings and then click on Gateway tab.

8> Now click on QuickPay and activate this plugin for this property and enter the required details.

	Merchant Id :- It is generated by QuickPay
	
    Agreement Id :- It is generated by QuickPay
	
    Payment Window Api key :- It is generated by QuickPay
